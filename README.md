# 🧠 Real-Time Chat App

A real-time chat application built with Node.js and Socket.IO.

## 🔧 Features
- Instant messaging
- Real-time updates using WebSockets
- Broadcast messages to all users

## 🛠️ Tech Stack
- Node.js
- Express.js
- Socket.IO
- HTML, CSS, JS

## 📂 Run Locally
```bash
git clone https://github.com/abhisparihar/ChatApp
cd chat-app
npm install
npm start
